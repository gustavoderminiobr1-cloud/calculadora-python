# Calculadora Python

Um projeto simples de calculadora em Python, usado para praticar o fluxo de contribuição com Git e GitHub.

## Funções disponíveis

- `somar(a, b)`
- `subtrair(a, b)`
- `multiplicar(a, b)`
- `dividir(a, b)`

## Usando a calculadora com botões (interface gráfica)

```bash
python calculadora_gui.py
```

Abre uma janela com botões de número, operações (+, -, *, /), `C` para limpar e `=` para calcular.

## Rodando os testes

```bash
pytest test_calculadora.py
```

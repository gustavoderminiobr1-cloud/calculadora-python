# Calculadora Python

Um projeto simples de calculadora em Python, usado para praticar o fluxo de contribuição com Git e GitHub.

## Funções disponíveis

- `somar(a, b)`
- `subtrair(a, b)`
- `multiplicar(a, b)`
- `dividir(a, b)`

## Usando a calculadora

```bash
python main.py
```

## Rodando os testes

```bash
pytest test_calculadora.py
```
